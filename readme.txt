PETSHOP - 2019

Para la ejecución del software es necesario iniciar el servidor desde Netbeans e ingresar a http://localhost:8080/PetShop

--------------------------------------------------------------------------------------------------------------------------------------------------------

Una vez en el index, se encuentran las siguientes opciones:
- Generar datos ficticios
- Generar datos ficticios desde TXT
- Empleados: Login para empleados
- Clientes: Login y formulario de registro para clientes

--------------------------------------------------------------------------------------------------------------------------------------------------------

Después de iniciar sesión se podrán ver las diferentes opciones para empleados o clientes, respectivamente.
